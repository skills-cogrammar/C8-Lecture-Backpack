Programming Languages Page Exercise Guide

Follow this step-by-step guide to build an interactive Programming Languages Page.

Objective

Create an interactive "Programming Languages" page that displays information about four popular programming languages using HTML, CSS, and Bootstrap.

Step 1: Set Up the HTML Structure

  1. Create the HTML File: Start by creating a file named index.html.
  2. Basic HTML Boilerplate: Add the basic HTML structure: 
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Programming Languages</title>
</head>
<body>

</body>
</html>
  3. Page Title and Header: Inside <body>, add a main title: 
    <header>
    <h1>Popular Programming Languages</h1>
</header>
  4. Programming Language Sections: Create a container <section> for language cards: 
    <section class="languages-container">
    <!-- Language cards will go here -->
</section>
  5. Individual Language Cards: Add one card with an image, heading, and description: 
    <div class="language-card">
    <img src="images/python-logo.png" alt="Python Logo">
    <h2>Python</h2>
    <p>Python is known for its readability and versatility.</p>
</div>

Step 2: Add Basic CSS for Layout and Styling

  1. Create a CSS File: Create a new file named styles.css in the same folder.
  2. Link the CSS: Link styles.css in <head>: 
    <link rel="stylesheet" href="styles.css">
  3. CSS for Basic Styling: Add global styles in styles.css: 
    body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    color: #333;
}
header {
    background-color: #0073e6;
    color: white;
    padding: 1rem;
}
  4. Language Card Styling: Style the card structure: 
    .language-card {
    width: 150px;
    border: 1px solid #ddd;
    border-radius: 8px;
    padding: 1rem;
    text-align: center;
}

Step 3: Integrate Bootstrap for Interactivity

  1. Link to Bootstrap: Add the Bootstrap CDN link in <head>: 
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha1/dist/css/bootstrap.min.css" rel="stylesheet">
  2. Convert Language Cards to Bootstrap Cards: Add Bootstrap card classes to each card, like this: 
    <div class="language-card card text-center">
    <img src="images/python-logo.png" class="card-img-top" alt="Python Logo">
    <div class="card-body">
        <h2 class="card-title">Python</h2>
        <p class="card-text">Python is known for readability.</p>
        <a href="https://www.python.org/" class="btn btn-primary">Learn More</a>
    </div>
</div>

Step 4: Final Touches and Testing

  1. Preview the Page: Open index.html in a browser to view your work.
  2. Adjust CSS: Modify styles in styles.css as needed to customize the look.

Good luck and enjoy building your interactive Programming Languages page!
